1) Посмотреть где я - pwd 
2) Создать папку - mkdir grup24
3) Зайти в папку - cd grup24
4) Создать 3 папки - mkdir 1p 2p 3p
5) Зайти в любоую папку - cd 2p
6) Создать 5 файлов (3 txt, 2 json) - touch 1.txt 2.txt 3.txt 4.json 5.json
7) Создать 3 папки - mkdir papka1 papka2 papka3
8. Вывести список содержимого папки - ls -la
9) + Открыть любой txt файл - vim 1.txt 
10) + написать туда что-нибудь, любой текст. - i
11) + сохранить и выйти. - клавиша "esc" :wq 
12) Выйти из папки на уровень выше - cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку. -  mv 2.txt papka1/2.txt
									    mv ./4.json ./papka1/4.json
									или mv ./{2.txt,4.json} ./papka1/{2.txt,4.json}
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. - cp ./1.txt ./papka1/1.txt
									   cp ./5.json ./papka1/5.json
15) Найти файл по имени - find 1.txt или find . -type f -name "1.txt"
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает. - cat 1.txt - содержимое в реальном времени
 										       - grep "какой-то текст" 1.txt - выводим что-то конкретное, а не все содержимое
17) вывести несколько первых строк из текстового файла - head -n 3 1.txt
18) вывести несколько последних строк из текстового файла - tail -n 3 1.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает. - less 1.txt
20) вывести дату и время - date
